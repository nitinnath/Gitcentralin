Algorithmic Trader Roadmap

You will learn how to create and maintain an automated trading system. This roadmap is for complete beginners. It has no prerequisites. 27 people marked this helpful

36 people marked this helpful

What you’ll learn

  • Computer Science And Programming Concepts
  • Python Programming Language
  • Interactive Brokers API
  • Scraping and Data Analysis
  • MySQL Database
  • Miscellaneous

Overview

Algorithmic traders are in demand today because almost 84% of all trades in the stock market are made automatically by computers. As an algorithmic trader, it will need to have strong understanding of application programming interface, data scraping, data analysis, database usage, etc.

This roadmap will guide you through some of the crucial technologies you need to learn in order to become a professioanl algorithmic trader.   This roadmap focuses on Python 3.6, MySQL 5.7 and Interactive Brokers API 9.73. Keep in mind that there are various other technology stacks you can choose from.

How to follow this roadmap?

  • Read on Python and see if it interests you. Otherwise, you can choose to follow any other roadmap.
  • If you want to learn the basics of computer science, you should start with the MIT course. It uses Python, but you should be able to pick up any language after taking it.
  • If you want to get started as soon as possible, you can start with any of the Python tutorials. After finishing whichever Python tutorial(s) you end up choosing, you should start with the “Investment Idiocy” tutorial.
  • MySQL is an open source database engine which you can integrate with your app. You can also look into Postgres or MariaDB.
  • If you found the roadmap is too difficult or it will simply takes too long to complete. You may hire our consultants to make your desired trading system for you.
